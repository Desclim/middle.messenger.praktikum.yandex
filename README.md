Веб-приложение мессенджера, разработанное в рамках курса от Яндекс Практикума.

Проект реализован как SPA (Single Page Application) без использования фреймворков.  
Навигация между страницами реализована через собственный роутер. Данные загружаются через API Яндекса


## Стек

- TypeScript
- Handlebars
- SCSS
- Vite
- ESLint
- Stylelint


## Функциональность

В приложении реализованы:

- регистрация пользователя;
- авторизация пользователя;
- выход из системы;
- просмотр страницы настроек пользователя;
- редактирование профиля;
- смена пароля;
- смена аватара;
- получение списка чатов;
- создание нового чата;
- добавление пользователя в чат;
- удаление пользователя из чата;
- роутинг между страницами без перезагрузки;
- защита приватных маршрутов для неавторизованных пользователей.


## Архитектура

Структура проекта:

src/
  api/           // классы для работы с HTTP API
  blocks/        // крупные составные блоки интерфейса
  components/    // переиспользуемые UI-компоненты
  controllers/   // слой бизнес-логики и связи API с приложением
  core/          // базовые классы: Block, Router, Store, HTTPTransport
  pages/         // страницы приложения
  services/      // форматирование и валидация
  styles/        // глобальные стили, переменные, миксины
  types/         // типы TypeScript
  utils/         // вспомогательные функции

## Установка

```bash
npm install
```

## Команды

Запуск в режиме разработки:

```bash
npm run dev
```

Проверка типов:

```bash
npm run typecheck
```

Линтинг TypeScript и стилей:

```bash
npm run lint
```

Сборка проекта:

```bash
npm run build
```

Сборка и запуск preview-сервера:

```bash
npm run start
```
